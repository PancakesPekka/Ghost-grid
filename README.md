> *Ghost Grid is a neon-drenched arcade shooter built on the bones of a 1978 classic — but the enemy has a memory.*

Every run you play is recorded. Enable **Ghost Replay** and your last attempt materialises beside you as a translucent blue ship, replaying every move you made. Beat your ghost. Beat your score. Beat yourself.

But the real threat isn't the invaders.

The **Commander** is an adaptive AI that studies your behaviour in real time. It tracks which side of the screen you favour, detects when you're holding back, and escalates accordingly. Silence a few columns of fire — then unleash a targeted burst aimed exactly where you're predicted to be. It taunts you on the way out. It remembers your peak. It deploys it against you.

---

## ✨ Features

| Feature | Description |
| :--- | :--- |
| 👻 **Ghost Replay** | Race your own previous run side by side |
| 🧠 **Adaptive Commander AI** | Learns your patterns, punishes them |
| 🎯 **Suppress &rarr; Burst** | Periods of silence followed by precision lock-on fire |
| ⚡ **Proximity Rage** | Enemies accelerate as they close in |
| 🔄 **Diagonal Drift** | Formations reshape on every wall bounce from wave 3 |
| ⚔️ **Split Shots** | Simultaneous dual-column fire from wave 5 |
| 💀 **Death Replay** | Your last 3 seconds replay in red before game over |
| 🛸 **Wave Formations** | Six distinct formations including V-Formation, Cluster Strike, Chevron, and Spread Assault |
| 🎵 **Procedural Audio** | No sound files, everything synthesised live entirely in code |

## 🚀 How to Play

**Play Online:**
Simply visit the [live GitHub Pages link](https://PancakesPekka.github.io/Ghost-grid/ghostgrid.html) to start playing immediately in your browser.

**Play Locally:**
1. Clone or download the repository to your local machine.
2. Double-click the `ghostgrid.html` file to open it in your favourite modern web browser.
3. **Survive.**

*No servers, no webpack, no installations, no dependencies. Just pure, single-file arcade survival.*
